# 注解的提取

## 核心结论

注解的提取依赖**反射机制**，前提是注解的 `@Retention` 策略为 `RUNTIME`。通过 `Class`、`Method`、`Field` 等反射对象的 `getAnnotation()`、`getAnnotations()` 系列方法获取注解信息。

## 深度解析

### 提取注解的核心 API

```java
// Class 对象上的注解方法
boolean isAnnotationPresent(Class<? extends Annotation> annotationClass)
<T extends Annotation> T getAnnotation(Class<T> annotationClass)
Annotation[] getAnnotations()
Annotation[] getDeclaredAnnotations()

// Method / Field / Constructor 同样有以上方法
```

### 从不同位置提取注解

```java
@Log("用户服务")
public class UserService {

    @Log("查询用户")
    public User getUser(int id) { return null; }

    @Deprecated
    public void oldMethod() { }
}

// 反射提取
public class AnnotationReader {
    public static void main(String[] args) throws Exception {
        // 1. 类上的注解
        Class<?> clazz = UserService.class;
        if (clazz.isAnnotationPresent(Log.class)) {
            Log log = clazz.getAnnotation(Log.class);
            System.out.println(log.value()); // 输出：用户服务
        }

        // 2. 方法上的注解
        Method method = clazz.getMethod("getUser", int.class);
        Log methodLog = method.getAnnotation(Log.class);
        System.out.println(methodLog.value()); // 输出：查询用户

        // 3. 获取所有注解（包括继承的）
        Annotation[] annotations = clazz.getAnnotations();
        for (Annotation ann : annotations) {
            System.out.println(ann.annotationType().getName());
        }

        // 4. 获取直接声明的注解（不包括继承的）
        Annotation[] declared = clazz.getDeclaredAnnotations();
    }
}
```

### 获取方法参数上的注解（JDK 8+）

```java
public void method(@Param("name") String name,
                   @Param("age") int age) { }

// 获取参数注解
Method method = clazz.getMethod("method", String.class, int.class);
Parameter[] parameters = method.getParameters();
for (Parameter param : parameters) {
    Param paramAnno = param.getAnnotation(Param.class);
    if (paramAnno != null) {
        System.out.println(param.getName() + " → " + paramAnno.value());
    }
}
```

### 反射注解的注意事项

1. **必须设置 RUNTIME**：否则 `getAnnotation()` 返回 null
2. **`getAnnotations()` vs `getDeclaredAnnotations()`**：前者包含继承的注解，后者只包含直接声明的
3. **性能开销**：反射获取注解有一定性能开销，框架通常会在启动时缓存注解信息
