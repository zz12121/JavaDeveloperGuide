# 字符串常量池

## 核心结论

字符串常量池（String Pool）是 JVM 的一块特殊内存区域，用于存储**字符串字面量**和通过 `intern()` 加入的字符串。目的是**节省内存**和**提高效率**，相同内容的字符串在常量池中只保留一份。

## 深度解析

### 常量池的位置变化

| JVM 版本 | 常量池位置 | 所属区域 |
|---------|----------|---------|
| JDK 6 | 永久代（方法区） | 堆外 |
| JDK 7 | 堆中 | 堆内 |
| JDK 8+ | 堆中 | 堆内（元空间取代永久代） |

JDK 7 将常量池从永久代移到堆中的原因：
- 永久代空间有限，容易 OOM
- 字符串使用频率高，放在堆中便于 GC 管理

### 常量池中字符串的存储

```java
// 1. 字面量直接存入常量池
String s1 = "hello";  // "hello" 放入常量池，s1 指向常量池
String s2 = "hello";  // 常量池已有 "hello"，s2 指向同一个引用
System.out.println(s1 == s2);  // true

// 2. 编译期常量拼接存入常量池
String s3 = "hel" + "lo";    // 编译期合并为 "hello"，指向常量池
System.out.println(s1 == s3);  // true

// 3. 变量拼接不进常量池
String a = "hel";
String b = "lo";
String s4 = a + b;            // 运行时拼接，在堆中创建新对象
System.out.println(s1 == s4);  // false
```

### new String() 与常量池

```java
String s1 = "hello";               // 常量池
String s2 = new String("hello");   // 堆中对象 + 常量池中的 "hello"
System.out.println(s1 == s2);      // false
System.out.println(s1 == s2.intern()); // true
```

### 常量池的 GC

- JDK 6：常量池在永久代，GC 不活跃，可能 OOM
- JDK 7+：常量池在堆中，当常量池中的字符串没有被任何强引用时，可以被 GC 回收

# new String("xxx") 创建几个对象

## 核心结论

`new String("xxx")` 创建 **1 或 2 个对象**：常量池中已有 "xxx" 时创建 1 个（堆中），常量池中没有 "xxx" 时创建 2 个（常量池中 1 个 + 堆中 1 个）。关键点：堆中的 String 对象内部的 `value` 数组引用的是常量池中字符串的 `value` 数组（共享）。

## 深度解析

### 创建过程分析
```java
String s = new String("hello");
```
分两步：
1. `"hello"` 是字面量 → 检查常量池是否已有 "hello"
    - 没有 → 在常量池中创建 "hello" 对象（1 个 String 类的实例对象）
    - 有 → 不创建
2. `new String(...)` → 在堆中创建一个新的 String 对象（1 个）

|情况|常量池已有 "hello"|常量池没有 "hello"|
|---|---|---|
|创建对象数|1（堆中）|2（常量池 + 堆中）|

### 引用关系
```
常量池:  "hello" (String对象，char[] = {'h','e','l','l','o'})
堆:      String对象 → value 引用 → 常量池中 "hello" 的 char[]
```
堆中的 String 对象并不复制 char[] 数组，而是引用常量池中的 char[]（共享）。

### 面试题拓展
```java
// 题目 1
String s1 = new String("hello");
String s2 = new String("hello");
System.out.println(s1 == s2);          // false（两个不同的堆对象）
System.out.println(s1.intern() == s2.intern()); // true（都是常量池中同一个引用）

// 题目 2
String s3 = "hello";
String s4 = new String("hello");
System.out.println(s3 == s4);          // false
System.out.println(s3 == s4.intern()); // true

// 题目 3
String s5 = new String("he") + new String("llo");
// 创建了 5 个对象：
// "he" 常量池1个 + new String("he") 堆1个 + "llo" 常量池1个 + new String("llo") 堆1个 + StringBuilder 拼接后的新 String 堆1个
// "helloworld" 不在常量池中
```
