# 方法引用

## 核心结论

方法引用（Method Reference）是 Lambda 表达式的简写形式：当 Lambda 表达式只是调用一个已有方法时，用 `类名::方法名` 或 `实例::方法名` 来替代。编译器自动推断参数和返回值类型，代码更简洁。

## 四种方法引用类型

| 类型 | 语法 | 等价 Lambda | 示例 |
|------|------|------------|------|
| **静态方法引用** | `ClassName::staticMethod` | `(args) → ClassName.staticMethod(args)` | `String::valueOf` |
| **实例方法引用（特定对象）** | `instance::instanceMethod` | `(args) → instance.instanceMethod(args)` | `System.out::println` |
| **实例方法引用（任意对象）** | `ClassName::instanceMethod` | `(arg, rest) → arg.instanceMethod(rest)` | `String::toUpperCase` |
| **构造器引用** | `ClassName::new` | `(args) → new ClassName(args)` | `ArrayList::new` |

### 静态方法引用

```java
// Lambda 形式
Function<Integer, String> lambda = i -> String.valueOf(i);

// 方法引用形式
Function<Integer, String> ref = String::valueOf;
ref.apply(42);  // "42"
```

### 特定对象的实例方法引用

```java
String prefix = "Hello, ";

// Lambda 形式
Consumer<String> lambda = s -> prefix.concat(s);

// 方法引用形式
Consumer<String> ref = prefix::concat;
ref.accept("World");  // "Hello, World"
```

### 任意对象的实例方法引用（最常用）

```java
// Lambda 形式
Function<String, String> lambda = s -> s.toUpperCase();

// 方法引用形式
Function<String, String> ref = String::toUpperCase;
ref.apply("hello");  // "HELLO"

// 在 Stream 中的典型用法
List<String> names = Arrays.asList("alice", "bob", "charlie");
names.stream()
     .map(String::toUpperCase)   // 任意对象引用
     .forEach(System.out::println); // 特定对象引用
```

### 与构造器引用的关系

详见 [[构造器引用]]。构造器引用 `ArrayList::new` 是方法引用的第四种形式。

## 细节规则

### 参数推断规则

```java
// 静态方法引用：参数直接传递
Function<String, Integer> f = Integer::parseInt;  // (String) → Integer

// 任意对象实例方法引用：第一个参数是方法调用者
Function<String, String> f = String::toUpperCase; // (String) → String
// 等价于 s -> s.toUpperCase()

// 特定对象实例方法引用：参数直接传递
Consumer<String> c = System.out::println;        // (String) → void
// 等价于 s -> System.out.println(s)
```

### 泛型方法引用

```java
// 方法引用可以与泛型结合
Function<String, Integer> f1 = Integer::parseInt;      // T → R
Supplier<List<String>> f2 = ArrayList::new;             // () → T

// Comparator 复合比较器
Comparator<Person> byName = Comparator.comparing(Person::getName);
Comparator<Person> byAge  = Comparator.comparing(Person::getAge);

// 多级排序
list.sort(byName.thenComparing(byAge));
```

## 方法引用 vs Lambda 对比

| 维度 | Lambda | 方法引用 |
|------|:------:|---------|
| 代码简洁性 | 需要写完整表达式 | 只需声明方法 |
| 可读性 | 复杂表达式更清晰 | 简单调用更简洁 |
| 适用场景 | 表达式复杂（有计算逻辑） | 纯方法调用 |
| 学习成本 | 通用语法 | 需要理解四种类型 |

```java
// 场景1：方法引用更好
list.stream().map(User::getName)          // 简洁清晰 ✅
list.stream().map(u -> u.getName())       // 啰嗦 ❌

// 场景2：Lambda 更好
list.stream().map(u -> u.getName().trim()) // 有计算逻辑 ✅
list.stream().map(String::trim)            // 无法表达取哪个对象的方法 ❌
```
