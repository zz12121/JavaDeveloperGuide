# 构造器引用

## 核心结论

构造器引用（Constructor Reference）是方法引用的一种特殊形式，使用 `类名::new` 语法。用于简化 Lambda 中创建对象的写法，根据函数式接口的签名自动匹配对应的构造器。

## 深度解析

### 基本用法

```java
// Lambda 形式
Supplier<User> s1 = () -> new User();
// 构造器引用
Supplier<User> s2 = User::new;

// 带参数的构造器
Function<String, User> f1 = name -> new User(name);
Function<String, User> f2 = User::new;

BiFunction<String, Integer, User> bf = User::new;
// 等价于 (name, age) -> new User(name, age)
```

### 匹配规则

构造器引用根据函数式接口的**参数个数**自动匹配构造器：

```java
public class User {
    public User() { }
    public User(String name) { }
    public User(String name, int age) { }
}

// 匹配无参构造器
Supplier<User> s = User::new;

// 匹配一个参数的构造器
Function<String, User> f = User::new;

// 匹配两个参数的构造器
BiFunction<String, Integer, User> bf = User::new;
```

### 数组构造器引用

```java
// Lambda
Function<Integer, String[]> arrayFactory = size -> new String[size];
// 数组构造器引用
Function<Integer, String[]> arrayFactory2 = String[]::new;

String[] arr = String[]::new.apply(10);  // 创建长度为 10 的 String 数组
```

### 在 Stream 中的使用

```java
// 将 List<String> 转为 List<User>
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
List<User> users = names.stream()
    .map(User::new)  // 构造器引用
    .collect(Collectors.toList());

// 将 Set 转为特定类型的集合
stream.collect(Collectors.toCollection(HashSet::new));
```

### 三个参数以上的构造器

如果构造器参数超过 2 个，标准库没有对应的三参数函数式接口，可以自定义：

```java
@FunctionalInterface
public interface TriFunction<A, B, C, R> {
    R apply(A a, B b, C c);
}

// 使用
TriFunction<String, Integer, String, User> factory = User::new;
```
