# 四大核心函数式接口

## 核心结论

`java.util.function` 包中定义了四大核心函数式接口：`Consumer<T>`（消费）、`Supplier<T>`（供给）、`Predicate<T>`（断言）、`Function<T,R>`（转换）。它们覆盖了"入-出"的四种组合，是 Stream API 和 Lambda 编程的基础。

## 深度解析

### `Consumer <T>` — 消费者（入无出）

```java
@FunctionalInterface
public interface Consumer<T> {
    void accept(T t);              // 接收参数，无返回
    default Consumer<T> andThen(Consumer<? super T> after); // 链式调用
}

// 使用
Consumer<String> print = s -> System.out.println(s);
print.accept("hello");  // 输出：hello

// 链式调用
Consumer<String> printUpper = s -> System.out.println(s.toUpperCase());
Consumer<String> printLen = s -> System.out.println(s.length());
printUpper.andThen(printLen).accept("hello"); // 先输出 HELLO，再输出 5
```

### `Supplier<T>` — 供给者（无入有出）

```java
@FunctionalInterface
public interface Supplier<T> {
    T get();  // 无参数，返回结果
}

// 使用
Supplier<Double> random = () -> Math.random();
System.out.println(random.get());  // 随机数

Supplier<List<String>> listFactory = ArrayList::new;
List<String> list = listFactory.get();  // 创建新 ArrayList
```

### `Predicate<T>` — 断言（入有出：boolean）

```java
@FunctionalInterface
public interface Predicate<T> {
    boolean test(T t);  // 接收参数，返回 boolean
    default Predicate<T> and(Predicate<? super T> other);
    default Predicate<T> or(Predicate<? super T> other);
    default Predicate<T> negate();
}

// 使用
Predicate<Integer> isPositive = x -> x > 0;
Predicate<Integer> isEven = x -> x % 2 == 0;

isPositive.test(5);                              // true
isPositive.and(isEven).test(4);                  // true
isPositive.or(isEven).test(-3);                  // true
isPositive.negate().test(5);                     // false
```

### `Function<T, R>` — 函数（入有出：R）

```java
@FunctionalInterface
public interface Function<T, R> {
    R apply(T t);  // 接收 T，返回 R
    default <V> Function<V, R> compose(Function<? super V, ? extends T> before);
    default <V> Function<T, V> andThen(Function<? super R, ? extends V> after);
    static <T> Function<T, T> identity();  // 恒等函数
}

// 使用
Function<String, Integer> strLen = String::length;
Function<Integer, String> intToStr = i -> "num=" + i;

strLen.apply("hello");          // 5
strLen.andThen(intToStr).apply("hello");  // "num=5"
intToStr.compose(strLen).apply("hello");   // "num=5"
Function.identity().apply("abc");           // "abc"
```

### 四大接口总结

| 接口 | 参数 | 返回值 | 典型用途 |
|------|------|--------|---------|
| `Consumer<T>` | T | void | forEach、日志输出 |
| `Supplier<T>` | 无 | T | 工厂方法、延迟计算 |
| `Predicate<T>` | T | boolean | filter 条件过滤 |
| `Function<T,R>` | T | R | map 类型转换 |
