## 核心结论

终端操作触发 Stream 流水线的**真正执行**，分为**非短路操作**（处理所有元素）和**短路操作**（不必处理所有元素即可返回结果）。一个 Stream 只能被消费一次。

---

## 深度解析

### 1. 收集操作

#### collect — 最灵活的终端操作
```java
// 收集为 List
List<String> list = stream.collect(Collectors.toList());

// 收集为 Set
Set<String> set = stream.collect(Collectors.toSet());

// 收集为 Map
Map<String, Integer> map = list.stream()
    .collect(Collectors.toMap(User::getName, User::getAge));

// 分组
Map<String, List<User>> groupByCity = users.stream()
    .collect(Collectors.groupingBy(User::getCity));

// 分区（二分组）
Map<Boolean, List<User>> partition = users.stream()
    .collect(Collectors.partitioningBy(u -> u.getAge() >= 18));

// 拼接字符串
String joined = list.stream().collect(Collectors.joining(", "));
```

### 2. 遍历与匹配

```java
// forEach
list.stream().forEach(System.out::println);

// anyMatch — 至少一个匹配（短路）
boolean hasLongName = list.stream().anyMatch(s -> s.length() > 10);

// allMatch — 全部匹配（短路）
boolean allPositive = nums.stream().allMatch(n -> n > 0);

// noneMatch — 全部不匹配（短路）
boolean noneNull = list.stream().noneMatch(Objects::isNull);

// findFirst — 返回第一个（短路）
Optional<String> first = list.stream().filter(s -> s.length() > 3).findFirst();

// findAny — 返回任意一个（短路，并行流中有优势）
Optional<String> any = list.parallelStream().findAny();
```

### 3. 聚合操作

```java
// count
long count = list.stream().count();

// min / max（返回 Optional）
Optional<Integer> min = nums.stream().min(Integer::compareTo);
Optional<User> oldest = users.stream().max(Comparator.comparing(User::getAge));

// sum / average / summaryStatistics（IntStream/LongStream/DoubleStream 专属）
int sum = nums.stream().mapToInt(Integer::intValue).sum();
IntSummaryStatistics stats = nums.stream().mapToInt(Integer::intValue).summaryStatistics();
// stats.getSum() / stats.getAverage() / stats.getMax() / stats.getMin() / stats.getCount()
```

### 4. 归约操作

#### reduce — 将流归纳为一个值

```java
// 两参数 reduce：初始值 + BinaryOperator
int sum = nums.stream().reduce(0, Integer::sum);

// 一参数 reduce：无初始值，返回 Optional
Optional<Integer> total = nums.stream().reduce(Integer::sum);

// 三参数 reduce：用于并行流（combiner 合并各线程结果）
int sum2 = nums.parallelStream().reduce(0, Integer::sum, Integer::sum);

// 求最大值
Optional<Integer> max = nums.stream().reduce(Integer::max);

// 字符串拼接
String result = list.stream().reduce("", String::concat);
```

### 5. 转换为数组

```java
String[] arr = list.stream().toArray(String[]::new);
int[] intArr = nums.stream().mapToInt(Integer::intValue).toArray();
```

### 6. 终端操作分类

| 类型 | 操作 | 是否短路 |
|------|------|----------|
| 收集 | `collect` | 否 |
| 遍历 | `forEach` | 否 |
| 匹配 | `anyMatch`/`allMatch`/`noneMatch` | **是** |
| 查找 | `findFirst`/`findAny` | **是** |
| 聚合 | `count`/`min`/`max`/`sum` | 否 |
| 归约 | `reduce` | 否 |
| 数组 | `toArray` | 否 |

> **关键理解**：短路操作在找到结果后立即停止处理剩余元素，类似 `||` 和 `&&` 的短路行为。
