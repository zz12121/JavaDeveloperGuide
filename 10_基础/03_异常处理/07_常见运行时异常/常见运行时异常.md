# 常见运行时异常（RuntimeException）

## 高频运行时异常速查

| 异常类 | 触发场景 | 典型代码 |
|--------|----------|----------|
| `NullPointerException` | 对 null 引用调用方法或访问属性 | `null.length()` |
| `ArrayIndexOutOfBoundsException` | 数组下标越界（< 0 或 ≥ length） | `arr[-1]` |
| `StringIndexOutOfBoundsException` | 字符串下标越界 | `"abc".charAt(5)` |
| `ClassCastException` | 类型强转失败 | `(String) new Integer(1)` |
| `NumberFormatException` | 字符串转数字格式错误 | `Integer.parseInt("abc")` |
| `ArithmeticException` | 算术错误（整数除零） | `1 / 0` |
| `IllegalArgumentException` | 方法参数非法 | 自定义检查抛出 |
| `IllegalStateException` | 对象状态不合法 | 迭代器未初始化就 next() |
| `UnsupportedOperationException` | 不支持该操作 | `Collections.unmodifiableList().add()` |
| `ConcurrentModificationException` | 迭代过程中修改集合 | 遍历 List 时 remove() |
| `StackOverflowError` | 无限递归导致栈溢出 | 方法无终止条件递归 |

## 详解高频异常

### 1. NullPointerException（NPE）

```java
String s = null;
s.length();             // NPE

// JDK 14+ 友好提示（Helpful NullPointerException）
// Cannot invoke "String.length()" because "s" is null
```

**防止 NPE 的方法：**
- `Objects.requireNonNull(obj, "msg")`
- `Optional<T>` 包装
- 先判空再使用

### 2. ConcurrentModificationException

```java
List<String> list = new ArrayList<>(Arrays.asList("a", "b", "c"));
for (String s : list) {
    if ("b".equals(s)) {
        list.remove(s);  // ❌ 会抛 ConcurrentModificationException
    }
}

// 正确方式1：Iterator.remove()
Iterator<String> it = list.iterator();
while (it.hasNext()) {
    if ("b".equals(it.next())) it.remove();  // ✅
}

// 正确方式2：removeIf（JDK 8+）
list.removeIf(s -> "b".equals(s));  // ✅
```

### 3. ClassCastException

```java
Object obj = Integer.valueOf(1);
String s = (String) obj;  // ❌ ClassCastException

// 防范：instanceof 检查
if (obj instanceof String) {
    String s = (String) obj;
}

// JDK 16+ Pattern Matching
if (obj instanceof String s) {
    // 自动转型
}
```

### 4. StackOverflowError

```java
// 无限递归
public void infiniteRecursion() {
    infiniteRecursion();  // StackOverflowError
}

// 防范：确保递归有终止条件
public int factorial(int n) {
    if (n <= 1) return 1;       // 终止条件
    return n * factorial(n - 1);
}
```
