# 泛型与反射结合

## 通过反射获取泛型信息

虽然运行时类型参数被擦除，但**声明在类/方法/字段上的泛型类型信息**会保留在字节码中（通过 `Signature` 属性），可通过反射 API 读取。

### 获取类的泛型父类

```java
public class UserRepo extends BaseRepository<User, Long> { }

// 获取泛型父类
Type superType = UserRepo.class.getGenericSuperclass();
if (superType instanceof ParameterizedType) {
    ParameterizedType pt = (ParameterizedType) superType;
    Type[] args = pt.getActualTypeArguments();
    System.out.println(args[0]);  // class User
    System.out.println(args[1]);  // class Long
}
```

### 获取字段的泛型类型

```java
public class Container {
    private List<String> items;
}

Field field = Container.class.getDeclaredField("items");
Type fieldType = field.getGenericType();

if (fieldType instanceof ParameterizedType) {
    ParameterizedType pt = (ParameterizedType) fieldType;
    Type[] args = pt.getActualTypeArguments();
    System.out.println(args[0]);  // class String
}
```

### 获取方法参数/返回值的泛型类型

```java
public class Service {
    public List<User> findAll() { return null; }
    public void save(Map<String, Object> params) { }
}

Method method = Service.class.getMethod("findAll");
Type returnType = method.getGenericReturnType();  // List<User>

Method saveMethod = Service.class.getMethod("save", Map.class);
Type[] paramTypes = saveMethod.getGenericParameterTypes();
// paramTypes[0] = Map<String, Object>
```

## 超类类型令牌（Super Type Token）

Jackson 和 Gson 获取泛型类型的核心技巧：

```java
// 匿名子类的泛型参数会被写入字节码
abstract class TypeReference<T> {
    public Type getType() {
        return ((ParameterizedType) getClass()
            .getGenericSuperclass())
            .getActualTypeArguments()[0];
    }
}

// 使用
TypeReference<List<String>> ref = new TypeReference<List<String>>() {};
Type type = ref.getType();  // java.util.List<java.lang.String>

// Jackson 实际用法
List<User> users = objectMapper.readValue(json, 
    new TypeReference<List<User>>() {});
```

## 反射创建泛型类的实例

```java
// 通用 Repository 基类，自动获取实体类型
public abstract class BaseRepository<T, ID> {
    
    private final Class<T> entityClass;
    
    @SuppressWarnings("unchecked")
    protected BaseRepository() {
        Type superType = getClass().getGenericSuperclass();
        ParameterizedType pt = (ParameterizedType) superType;
        this.entityClass = (Class<T>) pt.getActualTypeArguments()[0];
    }
    
    public T findById(ID id) {
        // 使用 entityClass 查询，如 session.get(entityClass, id)
        return null;
    }
}

// 子类无需手动传类型
public class UserRepository extends BaseRepository<User, Long> { }
```
