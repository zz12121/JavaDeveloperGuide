# 泛型接口（Generic Interface）

## 基本语法

```java
// 定义泛型接口
public interface Repository<T, ID> {
    T findById(ID id);
    List<T> findAll();
    void save(T entity);
    void delete(ID id);
}

// JDK 内置泛型接口示例
public interface Comparable<T> {
    int compareTo(T o);
}

public interface Iterable<T> {
    Iterator<T> iterator();
}
```

## 实现泛型接口的三种方式

### 方式一：实现时指定具体类型

```java
public class UserRepository implements Repository<User, Long> {
    @Override
    public User findById(Long id) { ... }
    
    @Override
    public List<User> findAll() { ... }
    
    @Override
    public void save(User user) { ... }
}
```

### 方式二：实现类也是泛型类（保留类型参数）

```java
public class GenericRepository<T, ID> implements Repository<T, ID> {
    @Override
    public T findById(ID id) { ... }
    
    @Override
    public List<T> findAll() { ... }
}

// 使用
GenericRepository<User, Long> repo = new GenericRepository<>();
```

### 方式三：实现类添加额外约束

```java
public class SortableRepository<T extends Comparable<T>> 
        implements Repository<T, Long> {
    // T 还额外要求可比较
}
```

## JDK 中常见泛型接口

| 接口 | 类型参数 | 说明 |
|------|----------|------|
| `Comparable<T>` | T | 对象自然排序 |
| `Comparator<T>` | T | 外部比较器 |
| `Iterable<T>` | T | 可迭代 |
| `Iterator<T>` | T | 迭代器 |
| `Callable<V>` | V | 有返回值的任务 |
| `Function<T,R>` | T,R | 函数式接口 |
| `Predicate<T>` | T | 断言接口 |
| `Supplier<T>` | T | 生产者 |
| `Consumer<T>` | T | 消费者 |

## 泛型接口的默认方法（JDK 8+）

```java
public interface Converter<F, T> {
    T convert(F from);
    
    // 默认方法也可以使用接口的类型参数
    default List<T> convertAll(List<F> list) {
        return list.stream()
                   .map(this::convert)
                   .collect(Collectors.toList());
    }
}
```
