# 泛型类（Generic Class）

## 基本语法

```java
// 单类型参数
public class Box<T> {
    private T value;
    
    public Box(T value) { this.value = value; }
    public T getValue() { return value; }
    public void setValue(T value) { this.value = value; }
}

// 多类型参数
public class Pair<A, B> {
    private A first;
    private B second;
    
    public Pair(A first, B second) {
        this.first = first;
        this.second = second;
    }
    public A getFirst() { return first; }
    public B getSecond() { return second; }
}
```

## 使用方式

```java
// 指定类型参数（推荐）
Box<String> strBox = new Box<>("Hello");
Box<Integer> intBox = new Box<>(42);

// JDK 7+ 菱形语法（右边类型推断）
Box<String> box = new Box<>("Hello");   // 不用重复写 <String>

// 原始类型（Raw Type，不推荐，会有警告）
Box rawBox = new Box("raw");  // 相当于 Box<Object>
```

## 有界类型参数

```java
// 上界：T 必须是 Number 或其子类
public class NumberBox<T extends Number> {
    private T value;
    
    public double doubleValue() {
        return value.doubleValue();  // 可以调用 Number 的方法
    }
}

// 多界：T 必须同时满足多个约束
public class MultiBox<T extends Comparable<T> & Serializable> {
    // T 既是 Comparable 又是 Serializable
}
```

## 泛型类的继承

```java
// 子类保留类型参数
public class StringBox extends Box<String> {
    // 固定了 T = String
}

// 子类也是泛型类
public class WrapBox<T> extends Box<T> {
    // 传递类型参数
}

// 子类扩展类型参数
public class DetailBox<T, U> extends Box<T> {
    private U extra;
}
```

## 注意事项

1. 类型参数不能使用基本类型（`Box<int>` 不合法，应用 `Box<Integer>`）
2. 静态成员不能使用类的类型参数（静态成员属于类而非实例）
3. 不能创建泛型数组（`new T[10]` 非法）
4. 不能实例化类型参数（`new T()` 非法）
