# 泛型方法（Generic Method）

## 基本语法

```java
// 格式：在返回值类型前声明类型参数 <T>
public <T> T identity(T t) {
    return t;
}

// 多类型参数
public <K, V> Map<K, V> mapOf(K key, V value) {
    Map<K, V> map = new HashMap<>();
    map.put(key, value);
    return map;
}

// 有界类型参数
public <T extends Comparable<T>> T max(T a, T b) {
    return a.compareTo(b) >= 0 ? a : b;
}
```

## 与泛型类的区别

| | 泛型类 | 泛型方法 |
|--|--------|---------|
| 声明位置 | 类名后 `class Box<T>` | 方法返回值前 `<T> T method()` |
| 作用范围 | 整个类（实例方法/字段） | 仅该方法 |
| 类型推断 | 实例化时指定 `new Box<String>()` | 调用时自动推断 |
| 是否可静态 | 类型参数不能用于静态成员 | 可以是静态方法 |

## 类型推断

```java
// 调用时编译器自动推断类型，无需显式指定
String s = identity("hello");    // T 推断为 String
Integer i = identity(42);        // T 推断为 Integer

// 极少数情况需要显式指定
List<String> list = Collections.<String>emptyList();
```

## 静态泛型方法

```java
public class Utils {
    // 静态泛型方法：用自己声明的 <T>，不依赖类的类型参数
    public static <T> List<T> listOf(T... elements) {
        return Arrays.asList(elements);
    }
    
    public static <T extends Comparable<T>> T min(List<T> list) {
        return list.stream().min(Comparator.naturalOrder()).orElseThrow();
    }
}

// 调用
List<String> strs = Utils.listOf("a", "b", "c");
String minStr = Utils.min(strs);  // "a"
```

## 实战：工具类泛型方法

```java
public class CollectionUtils {
    
    // 安全类型转换（替代强转）
    @SuppressWarnings("unchecked")
    public static <T> T cast(Object obj) {
        return (T) obj;
    }
    
    // 数组转 List（带类型安全）
    public static <T> List<T> toList(T[] arr) {
        return new ArrayList<>(Arrays.asList(arr));
    }
    
    // 集合转换
    public static <T, R> List<R> map(List<T> list, Function<T, R> mapper) {
        return list.stream().map(mapper).collect(Collectors.toList());
    }
    
    // 泛型 swap
    public static <T> void swap(T[] arr, int i, int j) {
        T temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```
