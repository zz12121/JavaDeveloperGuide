# 泛型通配符（Wildcard）

## 三种通配符

| 语法 | 名称 | 含义 |
|------|------|------|
| `<?>` | 无界通配符 | 任意类型 |
| `<? extends T>` | 上界通配符 | T 或 T 的子类（上界为 T） |
| `<? super T>` | 下界通配符 | T 或 T 的父类（下界为 T） |

## 上界通配符 `? extends T`（生产者/读取）

```java
// 可以读取，但不能写入（除了 null）
List<? extends Number> numbers = new ArrayList<Integer>();

Number n = numbers.get(0);   // ✅ 读取：至少是 Number
// numbers.add(1);           // ❌ 写入：不知道是哪种子类，不安全
// numbers.add(null);        // ✅ 唯一可以 add 的是 null

// 场景：求和（只需读取）
public double sum(List<? extends Number> list) {
    double sum = 0;
    for (Number n : list) sum += n.doubleValue();
    return sum;
}
// 可以传 List<Integer>、List<Double>、List<Long>
```

## 下界通配符 `? super T`（消费者/写入）

```java
// 可以写入 T 及其子类，但读取只能返回 Object
List<? super Integer> nums = new ArrayList<Number>();

nums.add(1);          // ✅ 写入 Integer：安全（Integer 是 Integer 的子类）
nums.add(2);          // ✅
Object o = nums.get(0); // ✅ 读取：只能用 Object 接收（不知道确切类型）
// Integer i = nums.get(0); // ❌ 编译错误

// 场景：填充数据（只需写入）
public void fill(List<? super Integer> list, int count) {
    for (int i = 0; i < count; i++) list.add(i);
}
// 可以传 List<Integer>、List<Number>、List<Object>
```

## 无界通配符 `?`

```java
// 相当于 ? extends Object
// 可以接收任意类型的 List，但只能做 null 以外的读取（返回 Object）
public void printList(List<?> list) {
    for (Object o : list) System.out.println(o);
}
// 可以传任意类型的 List

// 注意：List<?> 和 List<Object> 不同！
List<String> strings = new ArrayList<>();
// List<Object> objects = strings;  // ❌ 不兼容
List<?> wildcards = strings;        // ✅ 可以
```

## 读写能力总结

| 通配符 | 读取 | 写入 |
|--------|------|------|
| `<T>` | T | T |
| `<? extends T>` | T（可以） | ❌（不能，除 null） |
| `<? super T>` | Object（只能） | T 及其子类（可以） |
| `<?>` | Object（只能） | ❌（不能，除 null） |
