# static import 静态导入

## 核心结论

`static import` 用于导入类的**静态成员**（静态方法/静态常量），使其可以直接使用而无需类名限定。语法为 `import static 全限定名.静态成员`。
适用于频繁使用静态方法的场景（如 `Math.abs()`、`Collections.sort()`、`Assertions` 等），但**不可滥用**，过度使用会降低可读性。

---

## 深度解析

### 1. 基本语法

```java
// 传统方式
double result = Math.sqrt(25);
List<String> list = Collections.emptyList();

// static import 后
import static java.lang.Math.sqrt;
import static java.lang.Math.PI;
import static java.util.Collections.emptyList;

double result = sqrt(25);           // 无需 Math.
double area = PI * r * r;           // 无需 Math.
List<String> list = emptyList();    // 无需 Collections.
```

### 2. 两种导入方式

```java
// 导入指定静态成员（推荐）
import static java.lang.Math.sqrt;
import static java.lang.Math.max;

// 导入全部静态成员（通配符，慎用）
import static java.lang.Math.*;
```

> ⚠️ 通配符导入（`.*`）容易造成命名冲突，降低可读性，**不推荐**。

### 3. 使用场景

| 场景 | 示例 |
|------|------|
| Math 方法 | `import static java.lang.Math.*;` |
| Collections 工具方法 | `import static java.util.Collections.*;` |
| Assert 断言 | `import static org.junit.Assert.*;` |
| 标准常量 | `import static java.util.Calendar.JANUARY;` |

### 4. 命名冲突处理

```java
import static java.lang.Math.abs;
import static java.lang.StrictMath.abs;  // ❌ 编译错误，重复导入

// 解决方式：使用指定成员而非通配符，或保留类名限定
import static java.lang.Math.abs;
double a = abs(-1);          // 使用导入的
double b = StrictMath.abs(-1); // 保留类名
```

### 5. 与普通 import 的区别

| 对比项 | `import` | `import static` |
|--------|----------|-----------------|
| 导入目标 | 类/接口 | 静态方法/静态字段 |
| 使用方式 | `ClassName.member` | `member`（省略类名） |
| 适用场景 | 引用类型 | 频繁使用的静态工具方法 |

### 6. 最佳实践

- ✅ 对频繁使用的静态方法（如 `assertEquals`）使用
- ✅ 使用指定成员导入，**避免通配符**
- ❌ 不要为了"省几个字"而滥用
- ❌ 不要在同一个类中导入多个有同名静态成员的类

## 代码示例

```java
import static java.lang.Math.PI;
import static java.lang.Math.sqrt;
import static java.util.Collections.sort;
import static java.util.Collections.unmodifiableList;

public class StaticImportDemo {
    public static void main(String[] args) {
        double r = 5.0;
        double area = PI * r * r;    // 直接使用 PI，无需 Math.PI
        double d = sqrt(area);       // 直接使用 sqrt

        List<String> names = new ArrayList<>();
        sort(names);                  // 直接使用 sort
        List<String> safe = unmodifiableList(names);
    }
}
```
