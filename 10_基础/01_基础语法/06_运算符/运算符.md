# 运算符

## 先说结论

Java 运算符分为**算术、关系、逻辑、位运算、赋值和三元**六大类。
重点掌握：`&&` 短路特性、`%` 取模符号规则、`>>` 和 `>>>` 右移区别。

## 深度解析
### 算术运算符
### 算术运算符
| 运算符  | 说明       | 注意                          |
| ---- | -------- | --------------------------- |
| `+`  | 加法/字符串拼接 | 数值+字符串会拼接，不会算术运算            |
| `-`  | 减法       |                             |
| `*`  | 乘法       |                             |
| `/`  | 除法       | 整数除法截断小数部分，不四舍五入            |
| `%`  | 取模（取余）   | 结果符号与左操作数相同                 |
| `++` | 自增       | 前置 `++i` 先加后用，后置 `i++` 先用后加 |
| `--` | 自减       | 同上                          |

### 关系运算符
== 、`!=`、`>`、`<`、`>=`、`<=`，返回 boolean

### 逻辑运算符
### 逻辑运算符
| 运算符 | 说明 | 特点 |
|--------|------|------|
| `&&` | 短路与 | 左边为 false，右边不执行 |
| `\|\|` | 短路或 | 左边为 true，右边不执行 |
| `!` | 非 | 取反 |
| `&` | 逻辑与 | 两边都执行 |
| `\|` | 逻辑或 | 两边都执行 |
| `^` | 逻辑异或 | 相同为 false，不同为 true |

### 位运算符
### 位运算符
| 运算符 | 说明 |
|--------|------|
| `&` | 按位与 |
| `\|` | 按位或 |
| `^` | 按位异或 |
| `~` | 按位取反 |
| `<<` | 左移（低位补0，相当于 ×2^n） |
| `>>` | 带符号右移（高位补符号位） |
| `>>>` | 无符号右移（高位补0） |

### 三元运算符
`condition ? value1 : value2`

## 易错点/踩坑

- ❌ `10 / 3` = `3`（不是 3.33，整数除法截断）
- ❌ `-10 % 3` = `-1`（取模结果符号与左操作数相同，不是 Math.abs）
- ❌ `"a" + 1 + 2` = `"a12"`（从左到右拼接），`1 + 2 + "a"` = `"3a"`
- ❌ `&&` 短路特性：右边如果是有副作用的表达式（如 `i++`），可能不会执行
- ❌ `i = i++` 的结果：i 不变！（先取 i 的值保存，然后 i 自增，最后把保存的旧值赋回 i）

## 代码示例

```java
// ========== 整数除法 ==========
System.out.println(10 / 3);     // 3
System.out.println(10.0 / 3);   // 3.3333...

// ========== 取模 ==========
System.out.println(10 % 3);     // 1
System.out.println(-10 % 3);    // -1（符号与左操作数相同）

// ========== 字符串拼接 ==========
System.out.println("a" + 1 + 2);  // "a12"
System.out.println(1 + 2 + "a");  // "3a"

// ========== 短路 vs 非短路 ==========
int x = 1, y = 1;
// false && (x++ > 0) → x++ 不执行，x 仍为 1
// true || (y++ > 0)  → y++ 不执行，y 仍为 1

// ========== i++ 陷阱 ==========
int i = 0;
i = i++;  // i 仍为 0！（旧值 0 赋回 i）

// ========== 位运算 ==========
int a = 5;       // 0101
int b = 3;       // 0011
a & b;           // 0001 = 1
a | b;           // 0111 = 7
a ^ b;           // 0110 = 6
a << 1;          // 1010 = 10（相当于 ×2）
a >> 1;          // 0010 = 2（相当于 /2）

// ========== 三元运算符 ==========
int max = (a > b) ? a : b;
```

## 图解/流程

```
运算符分类
├── 算术运算符：+  -  *  /  %  ++  --
├── 关系运算符：==  !=  >  <  >=  <=
├── 逻辑运算符：&&  ||  !  &  |  ^
├── 位运算符：&  |  ^  ~  <<  >>  >>>
├── 赋值运算符：=  +=  -=  *=  /=  %=（复合赋值隐含强转）
└── 三元运算符：? :
```
