# 方法参数传递（值传递 vs 引用传递）

## 先说结论

**Java 只有值传递，没有引用传递。** 基本类型传递值的副本，引用类型传递引用地址的副本。
方法内修改引用指向的对象属性会影响原对象，但重新赋值不会影响原引用。

## 深度解析

### 基本类型传递
- 传递值的副本
- 方法内修改参数不影响原变量
- 本质：在栈帧中复制了一份值

### 引用类型传递
- 传递引用地址的副本（即指针的副本）
- 方法内通过引用修改对象**属性** → 影响原对象（因为指向同一个堆对象）
- 方法内**重新赋值**引用 → 不影响原引用（副本指向了新对象，原引用不变）

### 为什么说 Java 只有值传递？
- 对于基本类型：传的是值的副本，显然是值传递
- 对于引用类型：传的是地址值的副本，仍然是值传递
- "引用传递"的定义是：传引用本身（可以修改调用者的引用指向），Java 不支持

### 经典面试题：swap 为什么不能交换？
```java
void swap(int a, int b) {
    int temp = a; a = b; b = temp;
    // 只交换了副本，原变量不变
}
```

## 易错点/踩坑

- ❌ "Java 引用类型是引用传递" → 错！是"传递引用地址的值传递"
- ❌ 以为方法内修改引用类型参数不会影响外部 → 修改对象属性会影响
- ❌ 以为方法内修改引用类型参数一定影响外部 → 重新赋值不会影响
- ❌ String 是引用类型，但 String 不可变，方法内"修改"String 实际上是创建新对象

## 代码示例

```java
// ========== 基本类型：不影响原值 ==========
void change(int x) { x = 100; }
int a = 1;
change(a);
System.out.println(a); // 1（不变）

// ========== 引用类型：修改属性影响原对象 ==========
void changeName(Person p) { p.name = "张三"; }
Person person = new Person("李四");
changeName(person);
System.out.println(person.name); // 张三（被修改了）

// ========== 引用类型：重新赋值不影响原引用 ==========
void changeRef(Person p) { p = new Person("王五"); }
Person person2 = new Person("李四");
changeRef(person2);
System.out.println(person2.name); // 李四（不变！p 指向了新对象，person2 不受影响）

// ========== String 不可变 ==========
void changeStr(String s) { s = "world"; }
String str = "hello";
changeStr(str);
System.out.println(str); // hello（不变！String 不可变，s 指向了新对象）
```

## 图解/流程

```
基本类型传递：
  main 栈帧              change 栈帧
  ┌─────────┐           ┌─────────┐
  │ a = 1   │ ──复制──→ │ x = 1   │
  └─────────┘           │ x = 100 │（修改的是副本）
                        └─────────┘
  a 仍然是 1

引用类型传递（修改属性）：
  main 栈帧              change 栈帧
  ┌─────────┐           ┌─────────┐
  │ person  │──┐        │ p       │──┐
  └─────────┘  │        └─────────┘  │
               ▼                     ▼
          ┌────────────────────────────┐
          │ Person { name: "李四" }    │
          │         ↓ 修改             │
          │ Person { name: "张三" }    │ ← 两个引用指向同一对象
          └────────────────────────────┘

引用类型传递（重新赋值）：
  main 栈帧              change 栈帧
  ┌─────────┐           ┌─────────┐
  │ person  │──┐        │ p       │──┐（后来指向新对象）
  └─────────┘  │        └─────────┘  │
               ▼                     ▼
          ┌──────────┐         ┌──────────┐
          │ Person   │         │ Person   │
          │ {李四}   │         │ {王五}   │ ← 新对象，person 不受影响
          └──────────┘         └──────────┘
```
