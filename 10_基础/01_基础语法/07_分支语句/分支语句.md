# 分支语句（if/else、switch-case、switch 新特性）

## 先说结论

Java 分支语句分为 `if/else if/else` 和 `switch-case`。`switch` 适合**等值判断**（byte/short/int/char/String/枚举），`if` 适合**范围/条件判断**。
JDK 12+ 的 switch 表达式大大简化了写法。

## 深度解析

### if/else if/else
- 条件表达式必须是 boolean 值
- `else if` 是 `else { if }` 的简写
- 可以嵌套，但建议不超过 3 层

### switch-case 基础
- 支持的类型：`byte`、`short`、`int`、`char`、`String`（JDK 7+）、枚举（JDK 5+）
- 不支持：`long`、`float`、`double`、`boolean`
- case 值必须是**编译期常量**（字面量或 final 变量）
- 每个 case 需要 `break`，否则会发生**穿透**（fall-through）

### switch 新特性演进

| 版本 | 新特性 |
|------|--------|
| JDK 5 | 支持枚举 |
| JDK 7 | 支持 String |
| JDK 12 | switch 表达式（`break ->` 语法） |
| JDK 13 | 引入 `yield` 关键字返回值 |
| JDK 14 | switch 表达式正式版（箭头语法） |

### switch 表达式 vs switch 语句
- switch 语句：每个 case 用 `:` + `break`
- switch 表达式：用 `->` 箭头语法，可以有返回值，不会穿透

## 易错点/踩坑

- ❌ switch 穿透：忘记写 `break` 会继续执行下一个 case
- ❌ `switch(1.5)` 编译错误：不支持 float/double
- ❌ `case` 值不能重复：`case 1: ... case 1: ...` 编译报错
- ❌ switch 表达式中不能用 `break` 返回值，要用 `yield`
- ❌ switch 中定义变量要加大括号：多个 case 共享同一作用域

## 代码示例

```java
// ========== if/else ==========
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");  // 输出这个
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}

// ========== switch-case（传统写法）==========
String day = "MONDAY";
switch (day) {
    case "MONDAY":
        System.out.println("周一");
        break;
    case "TUESDAY":
        System.out.println("周二");
        break;
    default:
        System.out.println("其他");
}

// ========== switch 表达式（JDK 14+）==========
String result = switch (day) {
    case "MONDAY" -> "周一";
    case "TUESDAY" -> "周二";
    default -> "其他";
};

// switch 表达式 + 多行代码
String result2 = switch (day) {
    case "MONDAY", "FRIDAY" -> {
        System.out.println("工作日");
        yield "忙碌";  // yield 返回值
    }
    default -> "其他";
};

// ========== switch 穿透示例 ==========
int n = 2;
switch (n) {
    case 1:
        System.out.println("1");
    case 2:
        System.out.println("2");  // 输出
    case 3:
        System.out.println("3");  // 输出（穿透！）
        break;
    default:
        System.out.println("default");
}
```

## 图解/流程

```
switch-case 执行流程：

  ┌─────────────┐
  │ switch(表达式)│
  └──────┬──────┘
         ▼
  ┌─────────────┐
  │ 匹配 case 值 │
  └──────┬──────┘
         │
    ┌────┴────┐
    │ 匹配成功？│
    ├── 是 ──→ 执行对应代码块 ──→ break? ──→ 跳出 switch
    │              │
    │              └── 无 break ──→ 穿透到下一个 case
    │
    └── 否 ──→ 有 default? ──→ 执行 default ──→ 结束
                            └── 无 default ──→ 直接结束
```
