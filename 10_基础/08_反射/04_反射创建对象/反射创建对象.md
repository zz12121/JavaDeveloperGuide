# 反射创建对象

## 核心结论

反射创建对象有两种方式：`Class.newInstance()`（已弃用）和 `Constructor.newInstance()`（推荐）。推荐方式通过 `getDeclaredConstructor()` 获取构造器后调用 `newInstance()`，可以处理带参数构造器和私有构造器。

## 深度解析

### 方式一：Class.newInstance()（已弃用）

```java
// JDK 9 标记为 @Deprecated(forRemoval=true)
// 只能调用 public 无参构造器
User user = User.class.newInstance();
```

弃用原因：
- 只能调用 public 无参构造器
- 会直接向上抛出构造器中的所有异常（包括受检异常），无法精确定位
- 不支持带参数构造器

### 方式二：Constructor.newInstance()（推荐）

```java
// 1. 调用无参构造器
Constructor<User> constructor = User.class.getDeclaredConstructor();
User user = constructor.newInstance();

// 2. 调用带参构造器
Constructor<User> paramConstructor = User.class
    .getDeclaredConstructor(String.class, int.class);
User user2 = paramConstructor.newInstance("张三", 25);

// 3. 调用私有构造器（如单例模式）
Constructor<Singleton> privateConstructor = Singleton.class
    .getDeclaredConstructor();
privateConstructor.setAccessible(true);  // 突破 private 限制
Singleton instance = privateConstructor.newInstance();
```

### setAccessible(true) 的作用

- 默认情况下反射无法访问 private 成员，会抛出 `IllegalAccessException`
- `setAccessible(true)` 关闭访问权限检查，可以访问 private 构造器、方法、字段
- 底层是调用 `Field.setAccessible()`，JVM 不会做权限校验
- 如果存在 SecurityManager，可能仍然被拒绝

### Spring IoC 中的反射创建

```java
// Spring 创建 Bean 的简化流程
Class<?> clazz = Class.forName("com.example.UserService");
Constructor<?> constructor = clazz.getDeclaredConstructor();
constructor.setAccessible(true);
Object bean = constructor.newInstance();
// 然后通过反射注入依赖（Field.set()）
```

## 代码示例

```java
public class User {
    private String name;
    private int age;

    public User() { }

    private User(String name) {
        this.name = name;
    }

    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

// 反射创建
public class ReflectionDemo {
    public static void main(String[] args) throws Exception {
        // 无参构造
        User u1 = User.class.getDeclaredConstructor().newInstance();

        // 全参构造
        User u2 = User.class
            .getDeclaredConstructor(String.class, int.class)
            .newInstance("李四", 30);

        // 私有构造
        Constructor<User> c = User.class.getDeclaredConstructor(String.class);
        c.setAccessible(true);
        User u3 = c.newInstance("王五");
    }
}
```
