# 反射访问字段

## 核心结论

通过 `Field.get()` 和 `Field.set()` 可以在运行时读写对象的字段（包括私有字段）。对于私有字段需要先调用 `setAccessible(true)` 关闭权限检查。static 字段访问时对象参数传 null。

## 深度解析

### 基本用法

```java
public class User {
    private String name;
    public int age;

    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

// 反射访问字段
public class FieldAccessDemo {
    public static void main(String[] args) throws Exception {
        Class<?> clazz = User.class;
        Object user = clazz.getDeclaredConstructor(String.class, int.class)
            .newInstance("张三", 25);

        // 1. 读取 public 字段
        Field ageField = clazz.getField("age");
        int age = (int) ageField.get(user);
        System.out.println(age); // 输出：25

        // 2. 修改 public 字段
        ageField.set(user, 30);

        // 3. 读取 private 字段
        Field nameField = clazz.getDeclaredField("name");
        nameField.setAccessible(true);
        String name = (String) nameField.get(user);
        System.out.println(name); // 输出：张三

        // 4. 修改 private 字段
        nameField.set(user, "李四");
    }
}
```

### 访问 static 字段

```java
public class Config {
    private static String appName = "MyApp";
}

Field field = Config.class.getDeclaredField("appName");
field.setAccessible(true);

// static 字段，对象参数传 null
String value = (String) field.get(null);
field.set(null, "NewApp");
```

### 基本类型字段

```java
// Field 提供了基本类型的专用方法，避免装箱拆箱
Field intField = clazz.getDeclaredField("count");
intField.setAccessible(true);
int count = intField.getInt(user);     // 专用方法
intField.setInt(user, 100);            // 专用方法

// 也可以用通用方法（会自动装箱/拆箱）
int count2 = (int) intField.get(user);
intField.set(user, 100);
```

### 常量字段（final）的特殊处理

```java
// final 字段可以通过反射修改，但有两点注意：
// 1. 修改后如果编译器内联了常量值，其他地方看到的仍是旧值
// 2. static final 基本类型常量在编译期被内联，反射修改可能无效

Field field = Constants.class.getDeclaredField("MAX_SIZE");
field.setAccessible(true);

// 去除 final 修饰符
Field modifiersField = Field.class.getDeclaredField("modifiers");
modifiersField.setAccessible(true);
modifiersField.setInt(field, field.getModifiers() & ~Modifier.FINAL);

field.set(null, 200); // 修改 static final 字段
```
