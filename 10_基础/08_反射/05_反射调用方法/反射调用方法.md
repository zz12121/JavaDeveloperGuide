# 反射调用方法

## 核心结论

通过 `Method.invoke()` 可以在运行时动态调用对象的任意方法（包括私有方法）。`invoke()` 的第一个参数是目标对象（static 方法传 null），后续参数是方法实参。返回值是被调用方法的返回值。

## 深度解析

### 基本用法

```java
public class UserService {
    public String getName() { return "张三"; }
    private void log(String msg) { System.out.println("日志: " + msg); }
    public static int add(int a, int b) { return a + b; }
}

// 反射调用
public class MethodInvokeDemo {
    public static void main(String[] args) throws Exception {
        Class<?> clazz = UserService.class;
        Object target = clazz.getDeclaredConstructor().newInstance();

        // 1. 调用 public 方法
        Method getName = clazz.getMethod("getName");
        String result = (String) getName.invoke(target);
        System.out.println(result); // 输出：张三

        // 2. 调用 private 方法
        Method log = clazz.getDeclaredMethod("log", String.class);
        log.setAccessible(true);
        log.invoke(target, "反射调用私有方法"); // 输出：日志: 反射调用私有方法

        // 3. 调用 static 方法
        Method add = clazz.getMethod("add", int.class, int.class);
        int sum = (int) add.invoke(null, 3, 5); // static 方法传 null
        System.out.println(sum); // 输出：8
    }
}
```

### Method.invoke() 参数说明

```java
Object invoke(Object obj, Object... args)
```

| 参数 | 说明 |
|------|------|
| `obj` | 目标对象，static 方法传 null |
| `args` | 方法实参，基本类型需要自动装箱 |

### 基本类型参数和返回值

```java
// 基本类型参数需要传包装类型，会自动拆箱
Method setAge = clazz.getMethod("setAge", int.class);
setAge.invoke(user, 25);  // 25 自动装箱为 Integer，再拆箱为 int

// 基本类型返回值会自动装箱
Method getAge = clazz.getMethod("getAge");
int age = (int) getAge.invoke(user);
```

### 调用方法时的异常处理

```java
try {
    method.invoke(target, args);
} catch (InvocationTargetException e) {
    // 方法本身抛出的异常被包装为 InvocationTargetException
    // 真正的异常通过 e.getCause() 获取
    Throwable cause = e.getCause();
    cause.printStackTrace();
}
```

### 性能优化：缓存 Method 对象

```java
// 反射查找方法有性能开销，可以缓存
private static final Method GET_NAME_METHOD;
static {
    GET_NAME_METHOD = UserService.class.getMethod("getName");
}
// 后续直接使用缓存的 Method 对象
```
