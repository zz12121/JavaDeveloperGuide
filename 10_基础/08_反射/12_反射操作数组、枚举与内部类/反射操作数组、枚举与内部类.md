# 反射操作数组、枚举与内部类

## 反射操作数组

### 基本 API

| 方法 | 说明 |
|------|------|
| `Array.newInstance(componentType, length)` | 创建数组 |
| `Array.get(array, index)` | 获取元素 |
| `Array.set(array, index, value)` | 设置元素 |
| `Array.getLength(array)` | 获取长度 |

### 创建数组

```java
// 动态创建 String[] 数组
Class<?> stringArrayClass = Array.newInstance(String.class, 3).getClass();
System.out.println(stringArrayClass);  // class [Ljava.lang.String;

// 二维数组
int[][] matrix = (int[][]) Array.newInstance(int.class, 3, 4);

// 动态确定类型和长度
Class<?> componentType = Class.forName("java.lang.String");
Object array = Array.newInstance(componentType, 10);
```

### 访问和修改数组

```java
Object array = Array.newInstance(String.class, 3);

// 设置元素
Array.set(array, 0, "Hello");
Array.set(array, 1, "World");
Array.set(array, 2, "!");

// 获取元素
String s = (String) Array.get(array, 0);  // "Hello"

// 获取长度
int len = Array.getLength(array);  // 3
```

### 反射工具类：CollectionUtils 反序列化

```java
// 模拟 Jackson 反序列化时动态创建数组
public static Object stringArrayFromList(List<String> list) {
    Object array = Array.newInstance(String.class, list.size());
    for (int i = 0; i < list.size(); i++) {
        Array.set(array, i, list.get(i));
    }
    return array;
}
```

### 自动扩容

```java
// 反射实现 ArrayList 的 grow() 逻辑
Object oldArray = Array.newInstance(String.class, oldSize);
Array.set(oldArray, 0, "a");
Array.set(oldArray, 1, "b");

// 新数组 = 1.5 倍扩容
Object newArray = Array.newInstance(String.class, (int)(oldSize * 1.5));
System.arraycopy(oldArray, 0, newArray, 0, oldSize);
```

## 反射操作枚举

### 枚举的本质

```java
// 枚举在编译后实际是继承 java.lang.Enum 的类
public enum Status {
    PENDING,    // 实际是：public static final Status PENDING = new Status();
    APPROVED,
    REJECTED
}
```

### 获取枚举常量

```java
Class<Status> statusClass = Status.class;

// 获取所有枚举常量（declaredGetFields 是继承自 Enum 的）
Status[] constants = statusClass.getEnumConstants();
System.out.println(Arrays.toString(constants));  // [PENDING, APPROVED, REJECTED]

// 通过字符串创建枚举
Status s = Status.valueOf("PENDING");  // 等于 Status.PENDING

// 检查是否为枚举类
statusClass.isEnum();  // true
```

### 枚举的反序列化（Jackson / Fastjson）

```java
// Jackson 内部实现：通过 valueOf() 反序列化
Status s = Status.valueOf("APPROVED");

// 安全反序列化（防止非法值）
public static Status safeValueOf(String name) {
    try {
        return Status.valueOf(name);
    } catch (IllegalArgumentException e) {
        return null;  // 或返回默认值
    }
}
```

### 枚举字段反射

```java
// 枚举常量也是 Field
Field[] fields = statusClass.getDeclaredFields();
for (Field f : fields) {
    if (f.isEnumConstant()) {
        System.out.println(f.getName());  // PENDING, APPROVED, REJECTED
    }
}
```

## 反射操作内部类

### 获取内部类的 Class 对象

```java
// 静态内部类
Class<?> innerClass = Outer.Inner.class;
Class<?> innerClass2 = Class.forName("com.example.Outer$Inner");

// 成员内部类（实例级）
Class<?> innerClass3 = Class.forName("com.example.Outer$InnerInstance");

// 匿名内部类（无法通过名字获取）
Object obj = new Runnable() { };  // 只能从实例对象反推
obj.getClass().getEnclosingMethod();  // Runnable.run()
```

### getDeclaredClasses() vs getClasses()

```java
class Outer {
    class Inner {}
    static class StaticInner {}
}

class Derived extends Outer {}

// getDeclaredClasses()：仅当前类声明的内部类
Outer.class.getDeclaredClasses();    // [Inner, StaticInner]
// Derived 不声明 Inner，但...

// getClasses()：继承链上所有可见的内部类（包括继承的）
Derived.class.getClasses();           // [Inner] ✅ 通过继承获得
```

### 成员内部类的实例化

```java
// 成员内部类的实例化（必须通过外部类实例）
class Outer {
    class Inner {
        void greet() { System.out.println("hello"); }
    }
}

Outer outer = new Outer();
// 构造器参数必须是外部类实例
Outer.Inner inner = outer.new Inner();
```

### 通过反射实例化成员内部类

```java
Class<?> outerClass = Class.forName("com.example.Outer");
Class<?> innerClass = Class.forName("com.example.Outer$Inner");

// 1. 获取外部类实例
Object outerInstance = outerClass.newInstance();

// 2. 反射实例化内部类（new Inner(outerInstance)）
Constructor<?> ctor = innerClass.getDeclaredConstructor(outerClass);
ctor.setAccessible(true);
Object innerInstance = ctor.newInstance(outerInstance);
```

### 获取外部类实例

```java
// 从内部类实例反向获取外部类实例
Object innerInstance = ...;

// 方法1：通过 getEnclosingClass()
Class<?> outerClass = innerInstance.getClass().getEnclosingClass();

// 方法2：如果是成员内部类，通过 getDeclaredField("this$0")
Field enclosingField = innerInstance.getClass().getDeclaredField("this$0");
enclosingField.setAccessible(true);
Object outerInstance = enclosingField.get(innerInstance);
```

## 综合应用：动态 Bean 映射

```java
// 结合所有三种特殊类型的反射操作
public class ReflectiveMapper {

    public <T> T map(Object source, Class<T> targetClass) throws Exception {
        T target = targetClass.newInstance();

        for (Field field : source.getClass().getDeclaredFields()) {
            // 处理数组字段
            if (field.getType().isArray()) {
                Field targetField = findField(targetClass, field.getName());
                if (targetField != null) {
                    Object sourceArray = getFieldValue(source, field);
                    Object copiedArray = copyArray(sourceArray);
                    setFieldValue(target, targetField, copiedArray);
                }
            }
        }

        return target;
    }

    // 复制数组（演示 Array 反射操作）
    private Object copyArray(Object sourceArray) {
        int length = Array.getLength(sourceArray);
        Class<?> componentType = sourceArray.getClass().getComponentType();
        Object newArray = Array.newInstance(componentType, length);
        for (int i = 0; i < length; i++) {
            Array.set(newArray, i, Array.get(sourceArray, i));
        }
        return newArray;
    }
}
```
