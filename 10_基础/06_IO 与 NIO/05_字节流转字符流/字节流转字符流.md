# 字节流转字符流（InputStreamReader / OutputStreamWriter）

## 桥梁作用

`InputStreamReader` 和 `OutputStreamWriter` 是**字节流和字符流之间的桥梁**：

```
字节流                          字符流
InputStream ──→ InputStreamReader ──→ Reader
OutputStream ←── OutputStreamWriter ←── Writer
```

## InputStreamReader

将字节输入流按指定编码转换为字符输入流：

```java
// 构造方法
InputStreamReader(InputStream in)
InputStreamReader(InputStream in, String charsetName)
InputStreamReader(InputStream in, Charset charset)
```

### 工作原理

```
字节流: [0x48] [0x65] [0xE4] [0xB8] [0xAD]  (GBK: "He中")
                ↓ 解码（按指定编码）
字符流: ['H'] ['e'] ['中']
```

1. 从底层 `InputStream` 读取字节
2. 使用指定的 `Charset` 将字节解码为字符
3. 返回字符给调用者

## OutputStreamWriter

将字符输出流按指定编码转换为字节输出流：

```java
// 构造方法
OutputStreamWriter(OutputStream out)
OutputStreamWriter(OutputStream out, String charsetName)
OutputStreamWriter(OutputStream out, Charset charset)
```

### 工作原理

```
字符流: ['H'] ['e'] ['中']
                ↓ 编码（按指定编码）
字节流: [0x48] [0x65] [0xE4] [0xB8] [0xAD]  (GBK)
```

1. 接收字符
2. 使用指定的 `Charset` 将字符编码为字节
3. 写入底层 `OutputStream`

## 为什么需要它们

| 场景 | 说明 |
|------|------|
| 指定编码读取文件 | `new InputStreamReader(fis, "UTF-8")` |
| 处理网络数据 | 网络 TCP 是字节流，需要转为字符 |
| 不同编码转换 | 读 GBK 文件，写 UTF-8 文件 |

## 常用组合

### 指定编码读取文本

```java
try (BufferedReader br = new BufferedReader(
        new InputStreamReader(new FileInputStream("data.txt"), StandardCharsets.UTF_8))) {
    String line;
    while ((line = br.readLine()) != null) {
        System.out.println(line);
    }
}
```

### 指定编码写入文本

```java
try (BufferedWriter bw = new BufferedWriter(
        new OutputStreamWriter(new FileOutputStream("output.txt"), StandardCharsets.UTF_8))) {
    bw.write("中文内容");
}
```

### 编码转换（GBK → UTF-8）

```java
try (BufferedReader br = new BufferedReader(
        new InputStreamReader(new FileInputStream("gbk.txt"), "GBK"));
     BufferedWriter bw = new BufferedWriter(
        new OutputStreamWriter(new FileOutputStream("utf8.txt"), "UTF-8"))) {
    String line;
    while ((line = br.readLine()) != null) {
        bw.write(line);
        bw.newLine();
    }
}
```

## FileReader / FileWriter 的局限

`FileReader` 和 `FileWriter` 内部就是 `InputStreamReader` + `FileInputStream`，但**无法指定编码**（使用系统默认编码）：

```java
// FileReader 源码（简化）
public class FileReader extends InputStreamReader {
    public FileReader(String fileName) throws FileNotFoundException {
        super(new FileInputStream(fileName));  // 使用默认编码！
    }
}
```

所以涉及中文或跨平台时，推荐直接用 `InputStreamReader`/`OutputStreamWriter`。
