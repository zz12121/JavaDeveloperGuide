# 打印流（PrintStream / PrintWriter）

## 概述

打印流是**输出流**的装饰器，提供便捷的 `print()`、`println()`、`printf()` 方法。

| 类 | 继承 | 输出类型 | 特点 |
|----|------|---------|------|
| `PrintStream` | `FilterOutputStream` | 字节流 | `System.out` 就是它 |
| `PrintWriter` | `Writer` | 字符流 | 推荐，支持自动刷新 |

## PrintStream

```java
// System.out 就是 PrintStream
public final class System {
    public static final PrintStream out;
}
```

```java
PrintStream ps = new PrintStream("output.txt");
ps.println("Hello");
ps.printf("Name: %s, Age: %d", "Tom", 25);
ps.close();
```

**特点**：
- `print()` 方法**不抛 IOException**（内部捕获了）
- 输出字节流
- 可以指定编码

## PrintWriter（推荐）

```java
// 构造方式
PrintWriter pw = new PrintWriter("output.txt");
PrintWriter pw = new PrintWriter(new FileWriter("output.txt"));
PrintWriter pw = new PrintWriter(new BufferedWriter(new FileWriter("output.txt")));
```

```java
PrintWriter pw = new PrintWriter("output.txt");
pw.println("Hello");
pw.printf("Score: %.2f", 95.5);
pw.close();
```

**特点**：
- `print()` 方法**不抛 IOException**（通过 `checkError()` 检查）
- 自动刷新模式：`new PrintWriter(new BufferedWriter(fw), true)`（println 时自动 flush）
- 支持字符流，适合文本输出

## PrintStream vs PrintWriter

| 维度 | PrintStream | PrintWriter |
|------|------------|-------------|
| 继承 | `OutputStream` 子类 | `Writer` 子类 |
| 输出类型 | 字节流 | 字符流 |
| 自动刷新 | 只有 println/printf/格式 时 | 需要开启（构造器传 true） |
| 典型使用 | `System.out` | Servlet、文件写入 |

## 不抛异常的设计

打印流的 `print()`/`println()` 内部捕获了 `IOException`，不会向外抛出。需要通过 `checkError()` 判断是否出错：

```java
PrintWriter pw = new PrintWriter("output.txt");
pw.println("data");
if (pw.checkError()) {
    System.out.println("写入出错！");
}
```
