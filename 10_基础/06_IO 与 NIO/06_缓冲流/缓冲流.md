# 缓冲流（BufferedInputStream / BufferedReader）

## 为什么需要缓冲流

每次直接读写磁盘/网络，都要经过**系统调用**，开销大。缓冲流在内存中维护一个缓冲区，减少实际 I/O 操作次数。

```
无缓冲：每次 read() → 系统调用 → 磁盘
有缓冲：read() → 内存缓冲区（命中直接返回）
                    ↓ 缓冲区空了
                系统调用 → 磁盘（一次读一批）
```

## 四个缓冲流

| 缓冲流 | 包装对象 | 特殊方法 |
|--------|---------|---------|
| `BufferedInputStream` | `InputStream` | — |
| `BufferedOutputStream` | `OutputStream` | `flush()` |
| `BufferedReader` | `Reader` | `readLine()` |
| `BufferedWriter` | `Writer` | `newLine()` |

## 缓冲原理

### BufferedInputStream

```java
// 默认缓冲区大小 8192 字节
private static int DEFAULT_BUFFER_SIZE = 8192;

protected volatile byte buf[];  // 内部缓冲区
private int pos;                // 当前读取位置
private int count;              // 缓冲区有效数据长度
```

1. 首次 `read()` 时，从底层流一次性读取 8192 字节到 `buf[]`
2. 后续 `read()` 直接从 `buf[]` 返回，**不调用底层流**
3. `buf[]` 读完后，再从底层流填充一批

### BufferedOutputStream

```java
protected volatile byte buf[];
private int count;  // 缓冲区中当前数据量
```

1. `write()` 先写入 `buf[]`，不立即写入底层流
2. `buf[]` 满了，自动调用底层流的 `write()` 刷出
3. 也可以手动调用 `flush()` 立即刷出

### BufferedReader

```java
// 默认缓冲区大小 8192 字符
private static int defaultCharBufferSize = 8192;

// 核心方法：按行读取
public String readLine() {
    // 读取到 \n 或 \r 或 \r\n 时返回当前行内容
}
```

## 使用示例

### BufferedInputStream / BufferedOutputStream

```java
// 文件复制
try (BufferedInputStream bis = new BufferedInputStream(new FileInputStream("src.jpg"));
     BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream("dest.jpg"))) {
    byte[] buffer = new byte[8192];
    int len;
    while ((len = bis.read(buffer)) != -1) {
        bos.write(buffer, 0, len);
    }
    bos.flush();  // 确保缓冲区数据写入
}
```

### BufferedReader / BufferedWriter（最常用）

```java
// 按行读写文本文件
try (BufferedReader br = new BufferedReader(new FileReader("input.txt"));
     BufferedWriter bw = new BufferedWriter(new FileWriter("output.txt"))) {
    String line;
    while ((line = br.readLine()) != null) {
        bw.write(line);
        bw.newLine();  // 跨平台换行
    }
}
```

## flush() 的作用

```java
// close() 会自动调用 flush()
// 但在以下场景需要手动 flush：
PrintWriter pw = new PrintWriter(new BufferedOutputStream(new FileOutputStream("log.txt")));
pw.println("数据1");
pw.flush();  // 立即写入磁盘（不然可能在缓冲区中丢失）
```

## 性能对比

```java
// 无缓冲：逐字节读取（极慢）
// 缓冲：BufferedInputStream（快 100+ 倍）
// 手动缓冲：byte[] 数组（和 BufferedInputStream 差不多）

// 最佳实践：用 BufferedInputStream/BufferedReader，简洁且高效
```
