
# 句柄访问

## 先说结论

**句柄访问**是对象访问的一种方式，通过句柄池中的句柄间接访问对象。句柄包含**对象实例指针**和**类型数据指针**。优点是对象移动时只需修改句柄，缺点是多一次寻址。

## 深度解析

### 句柄访问原理

```
┌──────────────────────────────────────────────────────────────────┐
│                        句柄访问方式                                │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  栈帧（本地变量表）                                                │
│  ┌─────────────────────────────────────────────────────────────┐ │
│  │ reference（引用） ──────────────────────────────────────────┼─┼─│
│  │                        ↓                                    │ │  │
│  └─────────────────────────────────────────────────────────────┘ │  │
│                                     │                            │  │
│                                     ↓                            │  │
│  ┌─────────────────────────────────────────────────────────────┐ │  │
│  │                      句柄池（Handle Pool）                   │ │  │
│  │  ┌──────────────────────────────────────────────────────┐  │ │  │
│  │  │ 句柄1 ──┬─→ 对象实例指针 ──→ 堆中对象                   │  │ │  │
│  │  │         └─→ 类型数据指针 ──→ 方法区Klass                │  │ │  │
│  │  └──────────────────────────────────────────────────────┘  │ │  │
│  └─────────────────────────────────────────────────────────────┘ │  │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

### 句柄结构

| 组成部分 | 指向 | 说明 |
|----------|------|------|
| **对象实例指针** | 堆中对象 | 对象数据的实际地址 |
| **类型数据指针** | 方法区Klass | 类元数据 |

### 与直接指针的对比

| 特性 | 句柄访问 | 直接指针 |
|------|----------|----------|
| **间接层** | 有（句柄） | 无 |
| **对象移动** | 只需修改句柄 | 需更新所有引用 |
| **访问次数** | 两次寻址 | 一次寻址 |
| **速度** | 较慢 | 较快 |
| **稳定性** | 高（GC 移动对象不影响 reference） | 较低 |

## 易错点/踩坑

- ❌ 句柄访问需要额外的句柄池空间
- ❌ 每次访问对象多一次间接寻址
- ❌ 句柄池本身也需要管理
- ❌ HotSpot 默认使用直接指针，不是句柄访问

## 代码示例

```java
// 句柄访问示意（JVM 内部）
public class HandleAccessDemo {
    public static void main(String[] args) {
        Object obj = new Object();
        
        // 在 JVM 内部：
        // 1. reference 指向句柄池中的句柄
        // 2. 句柄包含两个指针
        // 3. 通过句柄访问对象
        
        System.out.println(obj.hashCode());
    }
}
```

```java
// 模拟句柄结构
class Oop {
    Object instance;   // 对象实例指针
    Klass  klass;      // 类型指针
}

// 访问对象
Oop handle = getHandle(obj);
handle.instance.field;  // 通过句柄访问
```

## 图解/流程

```
访问对象流程（句柄访问）

1. 读取 reference（栈帧）
       ↓
2. 读取句柄（句柄池）
       ↓
3. 从句柄获取对象实例指针
       ↓
4. 访问对象字段/方法
       ↓
5. 如需类型信息，从句柄获取类型指针
```
