
# 运行时常量池

## 先说结论

**运行时常量池**是方法区的一部分，是 Class 文件常量池在运行时的表示。存储字面量、符号引用、方法/字段引用，**JDK7 之前在永久代，JDK8+ 在元空间**。

## 深度解析

### 运行时常量池内容

```
┌──────────────────────────────────────────────────────────────────┐
│                    运行时常量池                                    │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │                      字面量                                  │ │
│  │  - 字符串字面量："hello"                                    │ │
│  │  - 整数：100, 200                                           │ │
│  │  - 浮点数：3.14                                            │ │
│  └────────────────────────────────────────────────────────────┘ │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │                    符号引用                                 │ │
│  │  - 类/接口：全限定名                                        │ │
│  │  - 方法：方法名 + 描述符                                     │ │
│  │  - 字段：字段名 + 描述符                                    │ │
│  └────────────────────────────────────────────────────────────┘ │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │                    解析结果                                 │ │
│  │  - 直接引用（内存地址）                                     │ │
│  │  - 类对象                                                  │ │
│  │  - 方法/字段入口                                            │ │
│  └────────────────────────────────────────────────────────────┘ │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

### 运行时常量池 vs Class 文件常量池

| 特性 | Class 文件常量池 | 运行时常量池 |
|------|-----------------|-------------|
| **位置** | .class 文件 | 方法区/元空间 |
| **时间** | 编译期 | 运行期 |
| **内容** | 符号引用 | 符号引用 + 直接引用 |
| **动态性** | 静态 | 可动态添加 |

### 运行时常量池的动态性

```java
// 运行时常量池可以动态添加内容

// 1. String.intern()
String s = new String("hello").intern();

// 2. 类的动态加载（反射）
Class.forName("com.example.Foo");

// 3. 动态生成类（ASM、Javassist）
```

## 易错点/踩坑

- ❌ 运行时常量池不等于字符串常量池（字符串常量池是它的一部分）
- ❌ JDK8+ 运行时常量池在元空间，不在堆中
- ❌ 运行时常量池可能抛出 OOM（元空间 OOM）
- ❌ 不是所有 Class 文件常量池内容都会进入运行时常量池

## 代码示例

```java
public class RuntimePoolDemo {
    public static void main(String[] args) {
        // 编译期常量（进入常量池）
        String s1 = "hello";      // 字面量
        int i = 100;              // 整数常量
        
        // 运行时常量池动态添加
        String s2 = new String("world");
        s2.intern();              // 添加到字符串常量池
        
        // 反射触发类加载
        try {
            Class<?> cls = Class.forName("java.lang.String");
        } catch (ClassNotFoundException e) {
        }
    }
}
```

```bash
# 查看类常量池
javap -v RuntimePoolDemo.class | head -50
```
