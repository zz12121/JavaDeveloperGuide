# 处理模型数据

## 先说结论

Spring MVC 提供多种方式向视图传递数据：`Model`、`ModelMap`、`ModelAndView`、`Map` 等。**Model 数据在请求结束后销毁**。

## 深度解析

### 模型接口体系

```
Model
    ↑
ModelMap
    ↑
ExtendedModelMap
    ↑
BindingAwareModelMap
```

### 接口对比

| 类型 | 说明 |
|------|------|
| Model | 接口，提供 addAttribute 方法 |
| ModelMap | 实现类，比 Model 更丰富 |
| ModelAndView | 同时包含模型和视图 |
| Map | 简化用法 |
| ModelAndView | 既包含模型数据，又包含视图信息 |

### 基本使用

```java
// Model
@GetMapping("/model")
public String withModel(Model model) {
    model.addAttribute("users", userService.list());
    return "user/list";
}

// ModelMap
@GetMapping("/modelmap")
public String withModelMap(ModelMap modelMap) {
    modelMap.addAttribute("users", userService.list());
    return "user/list";
}

// Map
@GetMapping("/map")
public String withMap(Map<String, Object> map) {
    map.put("users", userService.list());
    return "user/list";
}

// ModelAndView
@GetMapping("/mav")
public ModelAndView withModelAndView() {
    ModelAndView mav = new ModelAndView("user/list");
    mav.addObject("users", userService.list());
    return mav;
}
```

### 批量添加属性

```java
// 方式1：addAllAttributes
@GetMapping("/list")
public String list(Model model) {
    List<User> users = userService.list();
    Map<String, Object> data = new HashMap<>();
    data.put("users", users);
    data.put("total", users.size());
    model.addAllAttributes(data);
    return "user/list";
}

// 方式2：mergeAttributes
@GetMapping("/list")
public String list(Model model, @RequestParam(defaultValue = "10") int pageSize) {
    Map<String, Object> defaults = new HashMap<>();
    defaults.put("pageSize", pageSize);
    defaults.put("totalPages", 10);
    model.mergeAttributes(defaults);
    return "user/list";
}
```

## 易错点/踩坑

- ❌ @RestController 中使用 Model → 数据不会传递到视图
- ❌ Model 中的数据共享 → Model 是请求作用域，不共享
- ❌ Model 对象为 null → 注入失败，Spring MVC 自动创建

## 代码示例

### 分页场景

```java
@GetMapping("/page")
public String page(
    @RequestParam(defaultValue = "1") int pageNum,
    @RequestParam(defaultValue = "10") int pageSize,
    Model model
) {
    Page<User> page = userService.page(pageNum, pageSize);
    model.addAttribute("page", page);
    model.addAttribute("pageNum", pageNum);
    model.addAttribute("pageSize", pageSize);
    return "user/page";
}
```

### 条件查询

```java
@GetMapping("/search")
public String search(
    @RequestParam(required = false) String keyword,
    @RequestParam(required = false) String status,
    Model model
) {
    List<User> users = userService.search(keyword, status);
    model.addAttribute("users", users);
    model.addAttribute("keyword", keyword);
    model.addAttribute("status", status);
    return "user/list";
}
```
