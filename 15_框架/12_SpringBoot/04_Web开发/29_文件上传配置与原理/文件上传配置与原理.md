# 文件上传配置与原理

## 先说结论

Spring Boot 自动配置 `MultipartResolver`，支持文件上传。默认单文件最大 1MB，总请求最大 10MB。

## 深度解析

### 默认配置

```yaml
spring:
  servlet:
    multipart:
      enabled: true                    # 开启上传
      max-file-size: 10MB              # 单文件最大
      max-request-size: 100MB          # 总请求最大
      file-size-threshold: 2KB         # 内存阈值
      location: /tmp                    # 临时文件目录
```

### 上传原理

```
客户端 POST multipart/form-data
        ↓
MultipartResolver 解析请求
        ↓
MultipartFile 对象封装文件
        ↓
Controller 接收处理
```

## 易错点/踩坑

- ❌ 文件大小超限 → 413 Payload Too Large
- ❌ 未设置 `enctype="multipart/form-data"` → 文件为空
- ❌ 临时目录不存在 → 上传失败
