# 懒加载初始化

## 先说结论

Spring Boot 2.x 引入懒加载初始化（Lazy Initialization），Bean 在首次使用时才创建，而非启动时预创建，可加快应用启动速度，但会延迟错误发现时机。

## 深度解析

### 对比

```
┌─────────────────────────────────────────────────────────────┐
│                  立即加载 vs 懒加载                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   立即加载（默认）           懒加载                          │
│   ┌─────────────┐          ┌─────────────┐                 │
│   │ 启动时      │          │ 首次使用时  │                 │
│   │   ↓        │          │     ↓      │                 │
│   │ 创建所有Bean│          │   创建Bean │                 │
│   └─────────────┘          └─────────────┘                 │
│                                                             │
│   启动慢                        启动快                       │
│   错误提前暴露                  错误延迟暴露                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 配置方式

```yaml
# 全局配置
spring:
  main:
    lazy-initialization: true

# 或在代码中
new SpringApplication(App.class)
    .setLazyInitialization(true);
```

### @Lazy 注解

```java
// 类级别
@Lazy
@Component
public class MyService { }

// 方法级别
@Bean
@Lazy
public MyBean myBean() { }

// 注入时
@Bean
public MyConfig {
    @Lazy
    @Autowired
    private MyService myService;
}
```

### 优缺点

| 优点 | 缺点 |
|------|------|
| 启动速度快 | 错误延迟到运行时 |
| 减少内存占用 | 首次请求响应慢 |
| 按需创建 | 可能影响事务 |

## 易错点/踩坑

- ❌ @Lazy 在 @Bean 方法上使用位置错误
- ❌ 懒加载导致循环依赖检测不到
- ❌ 懒加载导致 @PostConstruct 执行时机不确定

## 代码示例

### 全局懒加载

```yaml
# application.yml
spring:
  main:
    lazy-initialization: true
```

### 局部懒加载

```java
@Configuration
public class AppConfig {
    
    // 这个 Bean 懒加载
    @Bean
    @Lazy
    public MyBean myBean() {
        return new MyBean();
    }
    
    // 这个 Bean 立即加载
    @Bean
    public MyBean2 myBean2() {
        return new MyBean2();
    }
}
```

### @Lazy 注入

```java
@Service
public class UserService {
    
    // 懒注入
    private final MyService myService;
    
    public UserService(@Lazy MyService myService) {
        this.myService = myService;
    }
}
```
